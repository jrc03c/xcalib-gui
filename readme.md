# Intro

`xcalib-gui` is a GUI for [`xcalib`](https://man.archlinux.org/man/xcalib.1.en). NOTE: Does not work with Wayland!

# Usage

To install:

```bash
git clone https://github.com/jrc03c/xcalib-gui
cd xcalib-gui
npm install
npm link
```

To use:

```bash
# (while still inside the cloned xcalib-gui repo folder)
electron .
```
